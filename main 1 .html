CSS (Cascading Style Sheets): Purpose: CSS is used to control the presentation, formatting, and layout of web pages. (JS)Java Script

JavaScript: Purpose: JavaScript is a programming language used to create dynamic and interactive web content. Function: It allows you to manipulate the content and behavior of web pages in real-time. JavaScript can be used for a wide range of tasks, from simple interactivity (e.g., form validation, button clicks) to complex operations (e.g., asynchronous data fetching, animations, and handling user inputs). .
